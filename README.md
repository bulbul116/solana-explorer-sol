# ☀️ Solana Explorer Backend
A lightweight Node.js API built with **Express** and **Helius RPC** to fetch real-time Solana wallet data.

## 🚀 Features
- **Wallet Info:** Fetches SOL balance and account details.
- **Transactions:** Retrieves the last 10 parsed transactions.
- **Health Check:** simple endpoint to verify server status.

## 🛠️ Tech Stack
- Node.js / Express
- Helius RPC (Solana Data)
- Axios (HTTP Requests)
